- **puma-pilot-presentation.html** - Responsive HTML presentation (v1 - basic version)

## How to View

1. **Open in browser:** Double-click the HTML file or open with any web browser
2. **Navigate slides:**
   - Scroll vertically (mouse/trackpad)
   - Arrow keys (↓ or → for next, ↑ or ← for previous)
   - Spacebar (next slide)
3. **Mobile viewing:** Works on any device, responsive design

## Features

- **PUMA branding:** Black background, white text, contextual colors
- **Responsive:** Adapts to phone, tablet, desktop screens
- **15 slides:** All content from SLIDE_DECK_OUTLINE.md
- **Print-ready:** Use browser print function to generate PDF

## Creating PDF

**Option 1: Browser print**
1. Open presentation in Chrome/Safari
2. Press Cmd+P (Mac) or Ctrl+P (Windows)
3. Select "Save as PDF"
4. Choose "Background graphics" option
5. Save

**Option 2: Export tools**
- Use Chrome's "Print to PDF" with background graphics enabled
- Or use dedicated HTML-to-PDF tools for higher quality
