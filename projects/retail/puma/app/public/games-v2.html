V2 Design Decisions

  • Type scale: 28px header / 20px scene / 18px titles / 15px body / 12px sub / 11px labels — clear 6-step hierarchy (SN #13 Hierarchy).
  • Weight as hierarchy: 900/800/700/600/500 — weight does the work, not just size (SN #12 Font Weight).
  • Numbered list: Giant muted "01/02/03" replaces icon boxes — type IS the decoration (SN #14 Titles & Body).
  • Minimal borders: Single bottom border instead of card borders — low visual noise (SN #45 Borders & Dividers).
  • Scene prominence: 20px/700 — the scenario text is the hero, not UI chrome (SN #28 Density).
  • Progress strip: Thin bar instead of dots — takes less visual space, clearer proportion (SN #25 Negative Space).
