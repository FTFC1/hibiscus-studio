# 2. User Flow

This document describes the step-by-step process for using the **Vehicle Dispatch Report Generator** web application.

### Step 1: Access the Web Application

Open your web browser and navigate to the application's URL: `https://vehicle-dispatch-api.onrender.com/`

### Step 2: Upload Your Report

On the main page, you will see an upload area. You can either:
*   **Drag and drop** your Excel (`.xls`, `.xlsx`) or CSV (`.csv`) report file directly onto the designated area.
*   **Click** the "Choose File" button to browse your computer and select your report.

### Step 3: Automatic Processing

Once you upload the file, the application will automatically begin processing your data. A loading indicator will be displayed during this time.

### Step 4: Review Results Summary

After processing is complete, the application will display a summary of your report. This includes key statistics like:
*   Total Vehicles Processed
*   Number of Brands
*   Unique Models
*   Total VINs Processed

You will also see a detailed breakdown of your brand portfolio.

### Step 5: Download the Cleaned Report

To get your standardized Excel report, click the **"Download Report"** button. The cleaned file will be saved to your computer.

### Step 6: Process Another File (Optional)

If you have another report to process, click the **"Process Another File"** button to return to the upload screen.
